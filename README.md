# Loja Tutti Frutti • Landing Page estilo Linktree

Cartão digital interativo (mobile-first) para bio do Instagram da **Loja Tutti Frutti**, com foco em conversão para WhatsApp.

## Stack
- React + TypeScript + Vite
- Tailwind CSS
- Framer Motion
- Lucide React

## Estrutura

```txt
src/
  components/
    CTAButton.tsx
    InfoCard.tsx
    ReviewCard.tsx
  App.tsx
  main.tsx
  styles.css
```

## Rodar localmente

```bash
npm install
npm run dev
```

## Build de produção

```bash
npm run build
npm run preview
```
